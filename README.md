# C++ Projects

A collection of small C++ applications demonstrating problem solving and programming fundamentals.

## Projects

### Banking Practice Program
Menu-driven banking simulator supporting deposits, withdrawals, and balance management.

### Credit Card Validator
Implementation of the Luhn algorithm for validating card numbers.

### Number Guessing Game
Randomized guessing game with attempt tracking.

### Console Calculator
Basic calculator supporting arithmetic operations.

### Hypotenuse Calculator
Uses the Pythagorean theorem to calculate a triangle's hypotenuse.
